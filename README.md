## 🧮 About NumPy
This project relies heavily on **[NumPy](https://numpy.org/)** (Numerical Python) for handling core data structures and mathematical operations. 

**Why we use it:**
* **Performance:** Leverages highly optimized, C-based arrays that are significantly faster than standard Python lists.
* **N-dimensional Arrays:** Utilizes `ndarray` objects for structuring and manipulating large datasets.
* **Mathematical Arsenal:** Employs built-in, comprehensive mathematical functions for fast operations on arrays without the need for complex loops.
